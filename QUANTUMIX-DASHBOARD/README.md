# QUANTUMIX Dashboard

This directory contains the frontend for the QUANTUMIX Dashboard.

## Previewing the Dashboard

To preview the dashboard, you can run a simple HTTP server from this directory.

1.  Make sure you are in the root of the repository.

2.  Start a Python HTTP server pointing to the `QUANTUMIX-DASHBOARD` directory:
    ```bash
    python3 -m http.server --directory QUANTUMIX-DASHBOARD 8000
    ```

3.  Open your browser and go to `http://localhost:8000` to view the dashboard.
